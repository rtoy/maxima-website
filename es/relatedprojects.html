Proyectos afines a Maxima

Esta página muestra una lista de paquetes de software que usan Maxima o que ayudan a trabajar con Maxima.

Interfaces de usuario para Maxima

Maxima on Android

Versión de Maxima para dispositivos móviles con Android.

Desarrollador: Yasuaki Honda.

wxMaxima

“wxMaxima es una GUI (interfaz gráfica de usuario, por sus siglas en inglés) multi-plataforma para el sistema de álgebra computacional Maxima, basada en wxWidgets. Provee una interfaz basada en menús y diálogos y una presentación agradable para las expresiones matemáticas.”

Desarrolladores: Andrej Vodopivec, Gunter Königsmann et al.

Imaxima and imath

“Imaxima.el provee soporte para interactuar con el sistema de álgebra computacional Maxima en un buffer de Emacs. Imaxima procesa la salida de Maxima con TeX e inserta la imagen resultante en el buffer.”

Desarrolladores: Jesper Harder y Yasuaki Honda.

TeXmacs

“GNU TeXmacs es un editor de texto científico libre tipo wysiwyw (lo que ves es lo que quieres, por sus siglas en inglés). El objetivo de este software es proveer un entorno de trabajo unificado y amigable para el usuario para la edición de documentos estructurados con diferentes tipos de contenido (texto, gráficos, matemáticas, contenido interactivo, etc.).”

Desarrollador: Joris van der Hoeven.

Kayali

“Kayali es un sistema de álgebra computacional basado en Qt que también puede ser usado como un avanzado reemplazo para KDE KCalc. Es esencialmente una GUI para Maxima (fácilmente extensible a otros CAS) y Gnuplot.”

Desarrollador: Abdulhaq Lynch.

Symaxx2

“Symaxx es una interfaz gráfica para el sistema de álgebra computacional Maxima (GPL).”

Desarrollador: Markus Nentwig.

Sistemas que usan Maxima como componente

Euler

“Euler es un sistema para trabajo numérico similar a Matlab, que cuenta con una GUI al estilo de cuaderno de notas (como Maple), con capacidades gráficas y un lenguaje de programación. Euler puede ser usado como una interfaz gráfica de Maxima. También puede intercambiar datos y expresiones con Maxima, ayudando a Maxima con los cálculos de índole numérico y a Euler con la evaluación de símbolos.”

Desarrollador: Rene Grothmann.

SAGE

“Sage es un sistema de software matemático de código abierto, distribuido bajo la licencia GPL. Desarrollado a partir de muchos paquetes de código abierto ya existentes: NumPy, SciPy, matplotlib, Sympy, Maxima, GAP, FLINT, R y muchos más.”

Desarrolladores: Más de 100 voluntarios.

STACK

“Sistema de código abierto para evaluación asistida por computador en matemáticas y áreas afines, con énfasis en evaluación formativa.”

Desarrolladores: Chris Sangwin et al.

WIMS

“La plataforma educativa WIMS ofrece una cornucopia muy rica de recursos y ejercicios o de acceso libre o para estudio personalizado...Los materiales de enseñanza/aprendizaje en WIMS son creados con la ayuda de software científico de código abierto de elevada calidad (Maxima, Gnuplot, Pari/Gp, Octave).”

Desarrolladores: Xiao Gang y otros.

Herramientas para incluir resultados de Maxima en otros sistemas

Maxiplot

Paquete Latex que proporciona entornos y comandos para incluir sesiones de Maxima y Gnuplot dentro de un documento, insertando los resultados y gráficas en el documento procesado. Necesita apenas descargar el fichero: maxiplot.sty y seguir las instrucciones en el Manual de Maxiplot (fichero fuente Latex: maxiplot.tex)

Desarrollador: José Miguel M. Planas

Jacomax

“Jacomax (Conexión de Java para Maxima) es una interfaz básica en Java para el sistema de álgebra computacional Maxima.”

Desarrollador: David McKain

Symbolic Equation Manipulator

“Edite ecuaciones directamente en la pantalla, usando el ratón, sin nunca cometer errores. Guíe el programa en cualquier manipulación algebraica complicada y deje que trate de los detalles tediosos. Muchas otras funcionalidades!”

Desarrollador: James Hart.

Extensión Algebra para Mediawiki

Permite incluir sesiones de Maxima dentro de páginas de Mediawiki. La instalación local de Maxima es después usada para obtener los resultados que son incorporados en la página Wiki.

Desarrollador: Markus Arndt.

Interfaces Web que ejecutan Maxima

Yamwi

“Un proyecto orientado al desarrollo de una interfaz web a Maxima escrita en PHP. El Centro de Supercomputación de Galicia aloja un entorno de cálculo Maxima+Yamwi en http://maxima.cesga.es.”

Desarrollador: Mario Rodríguez Riotorto.

Mathematical Assistant on Web

“Este sitio contiene una interfaz con el sistema de álgebra computacional Maxima y resuelve de forma automática un conjunto de problemas típicos de los cursos de matemáticas, incluyendo los pasos intermedios hasta llegar a la solución.”

Desarrolladores: Robert Marik y Miroslava Tihlarikova.

MaximaPHP

“Un programa en PHP para acceder a un servidor Maxima desde un sitio web en forma interactiva.”

Desarrollador: Bowo Prasetyo.

WebMathematics Interactive

“Sistema abierto de eLearning, basado en la web, sobre matemáticas para la escuela elemental y/o estudiantes universitarios, incluyendo funciones matemáticas básicas (resolución de ecuaciones, gráficas de funciones, derivación e integración simbólicas) y módulos temáticos.”

Desarrollador: Zoltan Kovacs.

Interactive Demos of Mathematical Computations

Demostraciones al vivo de varias funcionalidades de cálculo matemático.

Desarrolladores: Instituto de Matemáticas Computacionales de la Universidad Estatal de Kent.
