Proyectos relacionados

Interfaces de usuario para Maxima

Imaxima e imath

Desarrolladores: Jesper Harder, Yasuaki Honda.

“Imaxima.el provee soporte para interactuar con el sistema de álgebra computacional Maxima en un buffer de Emacs. Imaxima procesa la salida de Maxima con TeX e inserta la imagen resultante en el buffer.”

→ http://sites.google.com/site/imaximaimath/

Kayali

Desarrollador: Abdulhaq Lynch.

“Kayali es una sistema de álgebra computacional basado en Qt que también puede ser usado como un avanzado reemplazo para KDE KCalc. Es esencialmente una GUI para Maxima (y es fácilmente extendible para otros CAS) y Gnuplot.”

→ http://kayali.sourceforge.net/

LyX

Existe una manera de enviar comandos a Maxima desde LyX document processor. Es un poco diferente a como se hace en TeXmacs, pero parece ser funcional al menos para cuestiones básicas.

Un documento de ejemplo: http://maxima.sourceforge.net/lyx+maxima.lyx.

→ http://www.lyx.org/

Maxima on Android

Es posible ejecutar Maxima en un dispositivo Android.

→ https://sites.google.com/site/maximaonandroid/

Symaxx2

Desarrollador: Markus Nentwig.

“Symaxx es una interfaz gráfica para el sistema de álgebra computacional Maxima (GPL).”

→ http://symaxx.sourceforge.net/

TeXmacs

Desarrollador: Joris van der Hoeven.

“GNU TeXmacs es un editor de texto científico libre tipo wysiwym (lo que ves es lo que quieres, por sus siglas en inglés). El software anima a proveer un entorno de trabajo unificado y amigable para el usuario para la edición de documentos estructurados con diferentes tipos de contenido (texto, gráficos, matemáticas, contenido interactivo, etc.).”

→ http://www.texmacs.org/

wxMaxima

Desarrolladores: Andrej Vodopivec et al

“wxMaxima es una GUI (interfaz gráfica de usuario, por sus siglas en inglés) multiplataforma para el sistema de álgebra cimputacional Maxima basada en wxWidgets. Provee una interfaz basada en menus y diálogos y una buena salida para las expresiones matemáticas.”

→ http://wxmaxima.sourceforge.net/

Sistemas que usan Maxima como un componente en un amplio esquema

Euler

Desarrollador: Rene Grothmann.

“Euler es un sistema para trabajo numérico similar a Matlab, que cuenta con una GUI al estilo de cuaderno de notas (como Maple), con capacidades de graficación y un lenguaje de programación. Euler puede ser usado como una interfaz gráfica de Maxima. También puede intercambiar datos y expresiones con Maxima, ayudando a Maxima con los cálculos de índole numérico y a Euler con la evaluación de símbolos.”

→ http://mathsrv.ku-eichstaett.de/MGF/homes/grothmann/euler/

Red de Aprendizaje en línea con CAPA

Desarrollador: Gerd Kortemeyer.

“Compartiendo y usando materiales para aprendizaje y evaluación en línea a través de instituciones y disciplinas.”

Sistema de Álgebra Computacional (notas acerca de la integración de Maxima con LON-CAPA).

→ http://www.lon-capa.org/

MathDrag'n

Desarrollador: James Hart.

Página del proyecto MathDrag'n en SourceForge.

“MathDrag'n es una interfaz diseñada para ayudarte, el usuario, mantiene casi control total sobre el álgebra mientras se le avisa de sus errores. En contraste con la mayoría de sistemas de álgebra computacional, la filosofía de MathDrag'n's es que la interfaz de usuario está primero, y la facilidad de uso es la que tú quieras.”

→ http://mathdragn.squarespace.com/

Extensión de Álgebra para Mediawiki

Desarrollador: Markus Arndt.

“<Algebra> </Algebra> define una sesión Maxima. Cualquier línea terminada con un punto y coma es pasada a Maxima para ser evaluada. Para todas las otras lineas se aplica la sintaxis del wiki.”

→ http://meta.wikimedia.org/wiki/User:Mafs/Computer_algebra
