Загрузка

Вы можете загрузить файлы со страницы на SourceForge, там есть все версии Maxima.

Windows

Нужнен исполняемый установочный файл maxima-x.y.z.exe.

Linux

Для установки из RPM требуются по крайней мере два файла: maxima-x.y.z-n.i386.rpm с документами и скриптами и maxima-exec-<версия lisp>-x.y.z-n.i386.rpm с исполняемым образом Lisp. Еще может пригодиться maxima-xmaxima-x.y.z-n.i386.rpm — графическая оболочка Xmaxima.

Пакеты maxima и maxima-exec зваисят друг от друга, укажите оба команде rpm:

rpm -ivh maxima-x.y.z-n.i386.rpm maxima-exec-<версия lisp>-x.y.z-n.i386.rpm

Для установки из исходного кода берите maxima-x.y.z.tar.gz или maxima-x.y.z-n.src.rpm.

Заметки в README.lisps касаются реализаций Lisp, а в README.rpms — RPM.

Все платформы

Для пересборки из исходного кода нужен maxima-x.y.z.tar.gz.
