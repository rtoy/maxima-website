73.1 Introducción a solve_rec

El paquete solve_rec resuelve expresiones recurrentes lineales con coeficientes polinomiales.

Ejecútese demo(solve_rec); para ver una demostración sobre la utilización de este paquete.

Ejemplo: 

(%i1) load("solve_rec")$
(%i2) solve_rec((n+4)*s[n+2] + s[n+1] - (n+1)*s[n], s[n]);
                                    n
                 %k  (2 n + 3) (- 1)          %k
                   1                            2
(%o2)       s  = -------------------- + ---------------
             n     (n + 1) (n + 2)      (n + 1) (n + 2)
