Instalação em Windows

  1. Descarregue a versão mais recente, que atualmente encontra-se em: 5.45.1-Windows. Deverá descarregar o primeiro ficheiro na lista, que termina em win64.exe, a menos que tenha uma CPU muito antiga para a qual precisará a versão win32.exe.
  2. Execute o ficheiro que descarregou para iniciar o processo de instalação. Poderá demorar algum tempo, porque o programa instalador também instalará outros programas adicionais independentes do Maxima.
  3. Uma vez concluído o processo de instalação, procure o diretório onde o Maxima foi instalado, que será algo semelhante a: C:\\maxima_5.45.1. Nesse diretório deverá existir um subdiretório bin, que contém os programas instalados. Antes de executar qualquer um desses programas, verifique que o seu antivírus ignorará os programas nesse diretório. Se não tem certeza que assim será, faça então uma cópia do subdiretório completo em outra parte do disco. Acontece com frequência que, após a primeira execução do Maxima, o antivírus elimina o programa por considerá-lo um vírus. A cópia de segurança permitirá recuperar o ficheiro eliminado, sem ter que voltar a instalar Maxima.
  4. Antes de executar o Maxima, execute o programa lispselector que encontrará no mesmo diretório referido no passo anterior. Selecione "SBCL", que resolverá vários erros que podem surgir quando se usa a versão "Clisp" que vem ativa por omissão.
  5. Experimente as três interfaces do Maxima que encontrará no diretório bin: Maxima em linha de comandos, Xmaxima e Wxmaxima. Verifique que aparece o indicador inicial %i1 e que pode executar comandos básicos do Maxima.
  6. Para conferir que Maxima pode também aceder os programas gráficos que utiliza, execute os seguintes dois comandos, que deverão produzir gráficos: 
      plot2d (x, [x,0,1]);
  plotdf ([y,-x]);
