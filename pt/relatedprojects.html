Projetos relacionados com Maxima

Interfaces de utilizador para Maxima

Maxima para Android

Autor: Yasuaki Honda.

Versão de Maxima para dispositivos móveis com Android.

Sítio Web: https://sites.google.com/site/maximaonandroid/

wxMaxima

Autores: Andrej Vodopivec et al.

“wxMaxima é uma GUI (interface gráfica de utilizador, em inglês Graphical User Interface) multi plataforma para o sistema de álgebra computacional Maxima, baseada em wxWidgets. Fornece uma interface baseada em menus e diálogos e uma apresentação agradável das expressões matemáticas.”

Sítio Web: http://wxmaxima.sourceforge.net/

Imaxima e imath

Autores: Jesper Harder e Yasuaki Honda.

“Imaxima.el cria um buffer no editor Emacs para comunicar com o sistema de álgebra computacional Maxima. Imaxima processa a saída de Maxima com o sistema TeX e insere a imagem resultante nesse buffer.”

Sítio Web: http://sites.google.com/site/imaximaimath/

TeXmacs

Autor: Joris van der Hoeven.

“GNU TeXmacs é um editor de texto científico livre no estilo wysiwyw (o que vê é o que quer, em inglês what you see is what you want). O objetivo de este software é fornecer um ambiente de trabalho unificado e amigável para que o utilizador possa editar documentos estruturados com diferentes tipos de conteúdos (texto, gráficos, matemática, conteúdo interativo, etc.).”

Sítio Web: http://www.texmacs.org/

Kayali

Autor: Abdulhaq Lynch.

“Kayali é um sistema de álgebra computacional baseado em Qt que também pode ser usado como substituto avançado para KCalc no KDE. É essencialmente uma GUI para Maxima (facilmente extensível a outros CAS) e Gnuplot.”

Sítio Web: http://kayali.sourceforge.net/

Symaxx2

Autor: Markus Nentwig.

“Symaxx é uma interface gráfica para o sistema de álgebra computacional Maxima (GPL).”

Sítio Web: http://symaxx.sourceforge.net/Symaxx_1/

Sistemas que usam Maxima como parte de outro sistema

Euler

Autor: Rene Grothmann.

“Euler é um sistema para trabalho numérico semelhante a Matlab, que conta com uma GUI de estilo caderno de apontamentos (como Maple), com capacidades gráficas e uma linguagem de programação. Euler pode ser usado como interface gráfica para Maxima. Também pode trocar dados e expressões com Maxima, ajudando a Maxima com os cálculos de tipo numérico e a Euler com a manipulação de símbolos e expressões.”

Sítio Web: http://euler.rene-grothmann.de/

SAGE

Autores: Mais de 100 voluntarios.

“Sage é um sistema de software matemático de código aberto, distribuído sob a licença GPL. Desenvolvido a partir de muitos pacotes de código aberto já existentes: NumPy, SciPy, matplotlib, Sympy, Maxima, GAP, FLINT, R e muitos outros.”

Sítio Web: http://sagemath.org/

STACK

Autores: Chris Sangwin et al.

“Sistema de código aberto para avaliação assistida por computador em matemática e áreas afins, com ênfase na avaliação formativa.”

Sítio Web: http://stack.bham.ac.uk/

WIMS

Autores: Xiao Gang e outros.

“A plataforma educativa WIMS oferece um leque muito estenso de recursos e exercícios de acesso livre ou para estudo personalizado...Os materiais de ensino/aprendizagem em WIMS são criados com ajuda de software científico de código aberto de elevada qualidade (Maxima, Gnuplot, Pari/Gp, Octave).”

Sítio Web: http://wimsedu.info/?page_id=68

Ferramentas para incluir resultados de Maxima em outros sistemas

Maxiplot

Autor: José Miguel M. Planas

Pacote Latex que fornece macros e comandos para incluir sessões de Maxima e Gnuplot dentro de um documento, inserindo os resultados e gráficos no documento processado. Precisa apenas descarregar o ficheiro: maxiplot.sty e seguir as instruções no Manual de Maxiplot (ficheiro fonte Latex: maxiplot.tex)

Jacomax

Autor: David McKain

“Jacomax é uma interface básica em Java para o sistema de álgebra computacional Maxima.”

Sítio Web: https://www.wiki.ed.ac.uk/display/Physics/Jacomax

Symbolic Equation Manipulator

Autor: James Hart.

“Edite equações diretamente no ecrã, usando o rato, sem erros. Conduza o programa em qualquer manipulação algébrica complexa e deixe que trate dos pormenores. Muitas outras funcionalidades!”

Sítio Web: https://sourceforge.net/projects/equation/

Extensão Algebra para Mediawiki

Autor: Markus Arndt.

Permite incluir sessões de Maxima dentro de páginas de Mediawiki. A instalação local de Maxima é logo usada para obter os resultados que são incorporados na página Wiki.

Sítio Web: http://meta.wikimedia.org/wiki/User:Mafs/Computer_algebra

Interfaces Web que executam Maxima

Yamwi

Autor: Mario Rodríguez Riotorto.

“Projeto orientado ao desenvolvimento de uma interface web para Maxima escrita em PHP. O Centro de Supercomputação da Galiza aloja um sistema de cálculo Maxima+Yamwi em http://maxima.cesga.es.”

Sítio Web: http://yamwi.sourceforge.net

Online Algebra Calculator

A interface web Maxima online, originalmente baseada em Jacomax, permite aceder ao programa Maxima via Web.

Sítio Web: http://maxima-online.org

Mathematical Assistant on Web

Autores: Robert Marik e Miroslava Tihlarikova.

“Este sítio contem uma interface com o sistema de álgebra computacional Maxima e resolve de forma automática um conjunto de problemas típicos dos cursos de matemática, incluindo os passos intermédios até chegar à solução.”

Sítio Web: http://mathassistant.sourceforge.net/

MaximaPHP

Autor: Bowo Prasetyo.

“Programa em PHP para aceder a um servidor Maxima desde um sítio web de forma interativa.”

Sítio Web: https://sourceforge.net/projects/maximaphp/

WebMathematics Interactive

Autor: Zoltan Kovacs.

“Sistema aberto de e-learning, baseado na web, sobre matemática para a escola elementar e/ou estudantes universitários, incluindo funções matemáticas básicas (resolução de equações, gráficos de funções, derivação e integração simbólicas) e módulos temáticos.”

Sítio Web: https://sourceforge.net/projects/wmi/

Calc.Matthen.com

Autor: Matt Henderson.

Calculadora que aceita uma fórmula permitindo-lhe derivá-la, integrá-la, mostrar o seu gráfico, estendê-la, etc.

Sítio Web: http://calc.matthen.com/

Interactive Demos of Mathematical Computations

Autores: Instituto de Matemática Computacional da Universidade do Estado de Kent.

Demonstrações vivas de várias funcionalidades de cálculo matemático.

Sítio Web: http://icm.mcs.kent.edu/research/demo.html
