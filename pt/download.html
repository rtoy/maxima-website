Como descarregar Maxima

Podem obter-se ficheiros de instalação na Secção de ficheiros no sítio em Sourceforge. Essa secção contém apontadores para vários sistemas e versões diferentes de Maxima.

A secção Versões Recentes (à direita ou no fim desta página) mostra os números das versões mais recentes e clicando no número da versão, aparece o sumário das alterações nessa versão.

Também é possível experimentar o Maxima sem ter de o instalar, usando a interface Web: Maxima online interface.

Windows

Siga as instruções na secção: Instalação do Maxima em Windows

Macintosh

Atualmente, a forma mais fácil de instalar Maxima é usando Homebrew. Após instalar esse programa, utilize os seguintes comandos para instalar Maxima e Xmaxima:

   brew update
   brew install maxima
   brew install tcl-tk

Leia atentamente a informação após esses comandos, porque em alguns casos é necessário algum procedimento adicional para completar a instalação. O mesmo método pode ser utilizado para instalar Wxmaxima (brew install wxmaxima).

O sítio de Maxima em Sourceforge contém pacotes para versões antigas de Maxima, na secção Maxima-MacOS download section, e há um tutorial para instalar esse pacote em: tutorial to install Maxima in Mac OS, mas esse pacote não está a ser atualizado para versões mais recentes.

Linux RPM

Em distribuições de Linux baseadas em pacotes RPM, entre na secção Maxima-Linux e siga o apontador para a versão mais recente. Essa página terá vários pacotes RPM com nomes da forma: nome-versão.i386.rpm, onde a primeira parte é o nome do pacote e a segunda parte são os números que identificam a versão. O mínimo conjunto de pacotes que devem ser instalados são maxima e maxima-exec-clisp. Como esses dois pacotes dependem mutuamente um do outro, deverá instalá-los em simultâneo, com um único comando rpm:

rpm -ivh maxima-x.y.z-n.i386.rpm maxima-exec-clisp-x.y.z-n.i386.rpm

Clisp e Gnuplot deverão estar instalados no seu sistema.

Poderá também querer instalar maxima-xmaxima, que inclui uma interface gráfica e um programa gráfico necessário para que alguns comandos como plotdf e ploteq funcionem.

Os pacotes maxima-língua têm a tradução do manual em vários idiomas..

Android

As instruções de instalação e o URL para descarregar a aplicação encontram-se na página Maxima on Android.

Código fonte

Nos sistemas não incluídos na lista anterior, ou se quiser construir uma versão recente ou com uma variante diferente de Lisp, descarregue o código fonte na secção Maxima-source. Desempacote o conteúdo do arquivo e siga as instruções dadas no ficheiro INSTALL, que também pode ser consultado na cópia existente no servidor.

GIT

A versão mais recente de desenvolvimento pode ser descarregada do repositório GIT. Essa página mostra também o comando que pode ser usado para obter uma cópia completa do repositório. Após ter obtido a cópia do repositório, siga as instruções dadas no ficheiro INSTALL.git.
